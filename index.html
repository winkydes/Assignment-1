Responsive Web Design:

Responsive design is a common way for web programmers to design their application layout. Different elements in the webpage may have different appearances when the window size has changed. There are 3 examples of my responsive web design ideas. Try adjusting the widow size and see what it does!

1st Example: The container position

The "About me" and "courses" session is positioned horizontally when in large screen, vertically when in small screen.

2nd Example: Font size of my name in header

The font size of my name in header will change its size according to the screen width using CSS. (The profile picture will be gone when the window is small too!)
